============
Introduction
============

The extension allows you to add endpoints for SPARQL, SRU, and OAI-PMH to your
TYPO3 project. It is compatible to the Cultural Heritage Framework (CHF), which
provides various data models and a working Oxigraph container for SPARQL.

..  attention::

    The extension is not final yet. A lot of things may still change over the
    coming months.
